# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.4.x   | :white_check_mark: |
| < 0.4   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please DO NOT file a public issue.

Instead, send an email to: security@example.com

We will respond within 48 hours.

## Security Best Practices

1. Only use FLASHFLOOD on websites you own or have permission to test
2. Never share your proxy.txt file publicly
3. Use environment variables for sensitive data
4. Keep the tool updated to latest version
